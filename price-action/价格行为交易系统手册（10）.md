## 一、学习哲学：鲤鱼打挺的 10 步

文章用一个精妙的类比开场——学习鲤鱼打挺的 10 个步骤：

```
不会 → 能躺平 → 能立腰 → 能撑耳立腰 → 能蹬腿
→ 能画圆落地 → 能加高幅度 → 能蓄力推 → 能推蹲起 → 完成
```

**关键洞察**：

> "这里没有任何一步是'跳跃'的。每一步都只做一件事：把'不可能'拆成'已经发生过的事实'。"

### 映射到交易学习

```
会识别趋势 → 不会逆势做
会识别交易区间 → 知道上下边界
能识别边界 → 会识别边界是否突破
能识别突破情况 → 没突破前不做
能在区间没突破前不做 → 会在边界挂限价单
```

**交易失败的本质 = 跳步骤失败**。没有天赋，就多重复练习，重复到不再需要判断。

## 二、三大基础入场方式

### 2.1 突破单入场（Stop Order）

**在所有入场方式中胜率最高。**

| 维度 | 描述 |
|---|---|
| 机制 | 在信号棒的一端提前挂单，价格朝预期方向运动触发该价格时成交 |
| 核心优势 | 方向确认 + 市场验证——只有顺势突破才成交 |
| 避免的陷阱 | 不在价格反向时抄底摸顶 |
| 心理优势 | 入场后往往较快产生浮盈，更容易持仓 |

**劣势**：
- 相比限价单入场价格不够理想
- 快速突破时可能出现滑点

**Brooks 对应**（*Ranges* #1301 片段，得分 0.681）：

> "Although most trades should be entered on stops, when there is a strong trend it is safe to enter anytime."

Brooks 强烈推荐 Stop 单入场，因为"你被动量带着进入交易"。

### 2.2 收盘价/市价入场（Buy/Sell the Close）

| 维度 | 描述 |
|---|---|
| 机制 | 在强趋势棒收盘时立即用市价入场 |
| 适用场景 | 超大趋势棒、明确突破确认、趋势加速阶段 |
| 核心特征 | 无需等待，强调**紧迫感** |
| 紧迫感来源 | 强力 K 线的动能、对错过机会的担忧、对后续跟随的预期 |

**风险与限制**：

| 适合的情况 | 不适合的情况 |
|---|---|
| 大实体趋势棒 | 十字星、小实体 |
| 趋势方向明确 | 区间行情 |
| 信号极强 | 情绪波动较大的状态 |
| 交易者保持冷静理性 | 缺乏冷静判断时 |

### 2.3 限价单入场（Limit Order）

**目标**：获得更优惠的入场价格，优化风险回报比。需要更多判断和耐心。

#### 2.3.1 50% 回撤入场

在一波强劲脉冲之后，预期价格回调到该波动幅度的 50% 位置挂限价单入场。

#### 2.3.2 突破点回测入场

| 维度 | 描述 |
|---|---|
| 机制 | 在之前的突破位挂单等待价格回调 |
| 核心逻辑 | 突破点会从阻力变为支撑（或相反），吸引价格回测 |
| 适用条件 | 强力突破 + 明确确认 + 回测深度较浅 + 未破坏结构 |
| 不适合条件 | 突破力度弱、区间行情、回测过深可能破坏结构 |

> "市场也'记得'突破发生的位置，使得回测时该位置继续发挥作用。"

#### 2.3.3 关键位测试入场

在价格到达预设的重要价格位置时入场。

| 关键位 | 常见操作 |
|---|---|
| 昨日收盘价 | 从上方测试 → 支撑 → 可能做多；从下方测试 → 阻力 → 可能做空 |
| 昨日高点/低点 | 测试高点 → 可能做空/突破后做多；测试低点 → 可能做多/跌破后做空 |
| 今日开盘价 | 关键日内参考位 |
| 通道起点 | 通道起始的突破点 |
| 高潮棒极值 | 高潮 K 线的高低点 |

**两种操作方式**：
1. 在关键位置**预先挂限价单**
2. 等待价格到达关键位后**观察反应**，出现反转信号棒确认后再入场

#### 2.3.4 二次入场溢价规则

> **"好的成交价往往是坏的交易。"**

| 现象 | 解释 |
|---|---|
| 能获得"好价格"的入场 | 可能风险更高——深度回调到达关键位可能意味着趋势已改变 |
| 高质量的二次入场 | 通常伴随着更差的价格（溢价） |
| 核心教训 | 不能仅因为价格便宜就入场 |

### 入场方式对比总结

| 维度 | 突破单（Stop） | 收盘价/市价 | 限价单（Limit） |
|---|---|---|---|
| 入场时机 | 价格突破信号棒极值 | 强趋势棒收盘时 | 价格回调到预设位置 |
| 入场价格 | 较差 | 中间 | 更优惠 |
| 方向确认 | 强（市场验证） | 较强（动能信号） | 弱（逆小趋势） |
| 胜率 | 相对较高 | 中 | 相对较低 |
| 风报比 | 相对较低 | 中 | 相对较高 |
| 适用场景 | 大多数情况 | 强趋势棒加速 | 趋势明确、浅回调 |
| 风险 | 滑点 | 高点追多/低点追空 | 踏空、穿透、"好价格陷阱" |

## 三、高阶入场策略

### 3.1 陷阱价入场（Trap Entry）

**核心逻辑**：利用对手方被套的止损单力量。

| 陷阱类型 | 机制 | 操作 |
|---|---|---|
| **多头陷阱** | 多头在某价位做多后被套，止损单集中在某位置 | 价格触及止损位时大量卖单涌出 → 空头借此做空 |
| **空头陷阱** | 空头在某价位做空后被套，止损单集中在某位置 | 价格触及止损位时大量买单涌出 → 多头借此做多 |

**操作流程**：
1. 识别止损被触发的位置
2. 在该位置附近寻找反向入场机会
3. 利用被套交易者的止损力量推动价格

### 3.2 反向突破入场（Counter-Breakout Entry）

**核心逻辑**：预期某次突破会失败时进行逆向交易。

**判断突破可能失败的依据**：

| 信号 | 说明 |
|---|---|
| 突破力度弱 | 突破 K 线实体小、影线长 |
| 缺乏跟随 | 突破后没有连续同向 K 线 |
| 位于区间中 | 区间内突破容易失败 |
| 逆势突破 | 与主趋势方向相反 |
| 通道末端 | 通道顺向突破 75% 会失败 |

**操作示例（区间向上假突破后做空）**：
1. 识别交易区间
2. 价格尝试向上突破
3. 判断突破可能失败
4. 等待价格回到区间内
5. 在回到区间后的反转信号棒低点下方做空
6. 止损设在假突破高点上方

### 3.3 分批入场（Scaling In）

| 维度 | 描述 |
|---|---|
| 本质 | 仓位管理与入场结合的策略 |
| 机制 | 将总仓位拆分入场，价格反向运行时在更优价格补仓 |
| 前提 | 逻辑未改变的情况下于更优价位加仓 |
| 级别 | **仅限高阶交易者** |

**严正警告**：

> "分批入场可能演变为'越亏越加'，需要严格纪律与风险控制，新手容易滥用导致大额亏损。"

**不能分批入场的情况**：
- 逆势操作
- 在强趋势中对抗趋势方向
- 没有明确支撑阻力依据
- 只是因为"跌太多了"或"涨太多了"

**仓位管理规则**：
| 账户规模 | 最大分批次数 |
|---|---|
| 小账户 | 通常 2 次 |
| 大账户 | 可进行 3 次 |

## 四、Brooks 体系深化

### 4.1 Stop 单入场的权威支持

Brooks（*Reading* #651 片段，得分 0.677）明确了一个例外情况——可以不用 Stop 单而用 Limit 单：

> "There is one situation where you can often enter on a limit and not a stop. This occurs when you see a second entry after a trendline break and a strong reversal bar."

即：趋势线突破 + 强反转棒后的二次入场，可以用限价单在入场棒低点上方加仓。

### 4.2 限价单入场的规模

Brooks（*Ranges* #1282 片段，得分 0.667）：

> "There are about 10 reasonable stop entry setups and 20 or more limit order setups, although many are not obvious to inexperienced traders."

限价单的入场机会大约是 Stop 单的两倍，但风险更高，更难执行，因为方向与短期趋势相反。

### 4.3 分批入场的 Brooks 原话

Brooks（*Ranges* #921 片段，得分 0.686）：

> "Some traders trade smaller positions and scale in if the market continues against them; but only successful, experienced traders should ever attempt this."

**只有成功的、有经验的交易者才应该尝试分批入场。**

### 4.4 均线限价单的特殊价值

Brooks（*Ranges* #1301 片段）：

> "Entering at the moving average on a limit order is particularly good in stocks, which tend to be well behaved. This allows for a smaller risk and greater potential reward and essentially no change in the winning percentage."

在均线处用限价单入场，风险更小、潜在回报更大、胜率基本不变——特别适合股票。

## 五、入场方式选择决策树

```
判断市场状态
    ↓
├── 强趋势中（Always In 明确）
│   ├── 回调浅 → 限价单（均线/突破点回测/50% 回撤）
│   ├── 加速中 → 收盘价入场（惊喜 K 线收盘前）
│   └── 信号棒确认 → 突破单入场（信号棒极值 + 1 tick）
│
├── 弱趋势/通道中
│   ├── 通道边界 → 限价单（逆势，高风报比）
│   └── 旗形突破 → 突破单入场
│
├── 交易区间中
│   ├── 区间边界 → 限价单（高抛低吸）
│   ├── 假突破后 → 反向突破入场（陷阱策略）
│   └── 区间中间 → 不做！
│
└── 突破确认后
    ├── 强力突破 + 2 根跟随 → 突破单入场（顺势）
    └── 突破回测 → 限价单（突破点回测入场）
```

## 六、入场前的检查清单

### 通用检查清单
- [ ] 背景是什么？（趋势 / 区间 / 突破？）
- [ ] Always In 方向是否明确？
- [ ] 信号棒质量如何？（趋势棒 > 十字星；顺势可低质量）
- [ ] 信号棒是否在关键位置？

### Stop 单入场检查清单
- [ ] 信号棒是否已收盘？
- [ ] 入场价是否设在信号棒极值 + 1 tick？
- [ ] 止损是否设在信号棒反向极值 + 1 tick？
- [ ] 是否考虑了滑点？

### 收盘价入场检查清单
- [ ] 是否是超大实体趋势棒（惊喜 K 线）？
- [ ] 趋势方向是否明确？
- [ ] 是否在趋势初期（非末期高潮）？
- [ ] 情绪是否冷静？（情绪波动时不宜用）
- [ ] 是否不在区间中？

### 限价单入场检查清单
- [ ] 趋势方向是否明确？
- [ ] 回调深度是否合理（< 50%）？
- [ ] 回调是否有支撑迹象（影线、小实体）？
- [ ] 是否有踏空/穿透的风险意识？
- [ ] 便宜的入场价是否可能意味着趋势改变（"好价格陷阱"）？

### 分批入场检查清单（仅高阶）
- [ ] 交易逻辑是否未被破坏？
- [ ] 是否有明确的支撑/阻力依据？
- [ ] 是否不是在逆强趋势？
- [ ] 是否严格设定了最大分批次数？
- [ ] 是否设定了分批入场的最大亏损限额？

## 七、交易学习的阶梯模型

```
Level 1: 会识别趋势和区间
    ↓
Level 2: 会选择正确的入场方式（Stop / Limit / Close）
    ↓
Level 3: 会判断何时不做（区间中、高潮后、无信号）
    ↓
Level 4: 会使用陷阱策略（利用对手止损）
    ↓
Level 5: 会分批管理仓位（仅高阶）
```

---

## 八、一句话总结

> **Stop 单胜率最高、Limit 单风报比最好、Close 单最快。新手只用 Stop 单做主趋势；高阶在强趋势中用 Limit 单优化成本；分批入场是双刃剑，慎用。**
